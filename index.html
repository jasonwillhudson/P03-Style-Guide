.nav-link

Sample code:

.submit

Text Input Field with Label

CSS selector:

.text-input

Sample code:

<input type="text" id="text" name="text">
<label for="text">Label</label>

Rendered element:

Radio Button with Label

CSS selector:

.radio-button

Sample code:

<input type="radio" id="radio-button" name="label" value="label">
<label for="radio-button">Label</label>

Rendered element:
